Экспортируемые  функции модуля объекта

 

ПолучитьПараметры() - возвращает   структуру  параметров (из хранилища)

ПолучитьПараметр(НаименованиеПараметра) - в случае успеха возвращает  параметр  по имени,  иначе  "Неопределено"

ИзменитьПараметр(НовыеДанные) - НовыеДанные   Тип : Структура : {"НаименованиеПараметра, ЗначениеПараметра"} , при удачном изменении возвращает "Истина",   иначе  "Ложь"

УдалитьПараметр(Наименование) - при удачном удалении возвращает "Истина",   иначе  "Ложь"

 ПереименоватьПараметр(Ключ,НовИмя) - в случае успеха возвращает  "Истина",  иначе  "Ложь"

Экспортируемые  процедуры  и  функции общего  модуля  _37583_АлгоритмыСервер.

 

ПолучитьСсылкуСправочникАлгоритмы(Алгоритм)-  в качестве аргумента могут передаваться : Ссылка, УникальныйИдентификатор , Код  или Наименование. В случае успеха возвращает  Ссылку на элемент справочника,   иначе  "Неопределено"

ПолучитьКодАлгоритма(ПредставлениеАлгоритма) - возвращает  код  переданного алгоритма

ВыполнитьПроцедуру(ПредставлениеАлгоритма, ДополнительныеПараметры=Неопределено) -  ДополнительныеПараметры Тип : Структура : ( Если <> Неопределено  в нее записываются дополнительные  свойства  {Отказ,СообщениеОбОшибке,Результат}  )

ВыполнитьФункцию(ПредставлениеАлгоритма,ДополнительныеПараметры=Неопределено) - возвращает  Соответствие {Отказ,СообщениеОбОшибке,Результат , ...}

@Процедура -  ВыполнитьПроцедуру   пример: @Процедура(Ссылка|Код|Наименование,[СтруктураДопПараметров]) - в СтруктураДопПараметров возвращается  структура  {Отказ,СообщениеОбОшибке,Результат}

@Функция ВыполнитьФункцию пример: Результат=@Функция(@ВнешАлгоритм,НовыйСтруктура("ПараметрФункцииТипСтрока","Строка"))["Результат"];   - Результат - тип Соответствие  {Отказ,СообщениеОбОшибке,Результат , ...}  Напр.  Дата = @Функция(@алгоритм)#Результат;
